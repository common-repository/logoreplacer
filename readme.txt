== Description ==

This plugin replace WP default admin/login/register page.
The technique used is pretty brutal but the best currently possible, the plugin replaces the modified files automatically.
To make the plugin run the web server must be able to write to the wp-admin directory. 
more info @ http://www.bisontech.net

== Installation ==

1. Download the .zip file of logoReplacer from the WordPress plugins directory.
1. Unzip it into temp directory, edit all the file in logos directory under plugin directory and save with same format.
1. Upload logoReplacer direcotry  into the /wp-content/plugins/
1. Activate the plugin through the "Plugins" menu in WordPress.
